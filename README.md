# YouTube Clone

This project is a simple clone of the YouTube interface, built using HTML, CSS, and JavaScript. It aims to replicate the basic structure and functionality of the popular video-sharing platform.

## Project Structure

- `index.html`: The main page of the site, containing the HTML structure, including the header, navigation, main content, and footer.
- `css/style.css`: Contains styles for the site, defining the appearance of elements such as fonts, colors, margins, and layout to make the site resemble YouTube.
- `js/main.js`: Contains JavaScript code that adds interactivity to the site, including event handling for button clicks and dynamic content updates.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/youtube-clone.git
   ```
2. Navigate to the project directory:
   ```
   cd youtube-clone
   ```

## Usage

1. Open `index.html` in your web browser to view the site.
2. Interact with the elements to explore the functionality.

## Features

- Responsive design that adapts to different screen sizes.
- Basic navigation similar to YouTube.
- Placeholder content for videos and channels.
